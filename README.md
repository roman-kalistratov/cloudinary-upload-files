## Configuration
https://cloudinary.com/
- Add a `.env` file to the root directory.

# server

- Add a `.env` file to the root directory.

### Setup `.env` file

```shell
PORT = 5000
CLOUD_NAME = "Your cloud_name"
API_KEY = "Your api_key connection"
API_SECRET = "Your api_secret"
```

### Build and start the app

server

```shell
cd server
npm install
npm start
```

client:

```shell
cd client
npm install
npm start
```
